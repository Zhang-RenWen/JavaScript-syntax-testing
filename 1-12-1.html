Sets

在數學中，集合（Set）是一個基本的概念，是一群具備某種共同特性的對象的集結。這些對象稱為集合的元素（或成員）。集合的元素可以是任何東西，包括數字、字母、符號、物體，甚至是其他集合。

集合的表示方法

  1. 列舉法：直接將集合的所有元素列舉出來，用大括號表示。例如，集合 A 包含 1、2 和 3，可以寫作 A = {1, 2, 3}。
  2. 描述法：用一個性質或條件來描述集合中的元素。例如，集合 B 包含所有小於 5 的正整數，可以寫作 B = {x | x 是小於 5 的正整數}。

集合的基本概念

  1. 空集：不包含任何元素的集合，表示為 ∅ 或 {}。
  2. 子集：如果集合 A 的所有元素都是集合 B 的元素，則 A 是 B 的子集，記作 A ⊆ B。
  3. 真子集：如果集合 A 是集合 B 的子集，且 A ≠ B，則 A 是 B 的真子集，記作 A ⊂ B。
  4. 全集：包含所有考慮中的元素的集合，通常用 U 表示。

集合的運算

  1. 聯集（Union）：集合 A 和 B 的聯集包含 A 或 B 中的所有元素，記作 A ∪ B。
  2. 交集（Intersection）：集合 A 和 B 的交集包含 A 和 B 中的共同元素，記作 A ∩ B。
  3. 差集（Difference）：集合 A 和 B 的差集包含在 A 中但不在 B 中的元素，記作 A - B 或 A \ B。
  4. 補集（Complement）：在全集 U 中，集合 A 的補集包含不在 A 中的所有元素，記作 A' 或 U - A。

集合的特性

  • 元素唯一性：在集合中，每個元素都是唯一的，沒有重複的元素。
  • 無序性：集合中的元素沒有特定的順序。

Symbols

在數學中，字母 𝑁 、 𝑍 、 𝑄 、 𝑅 、 𝐶 用來表示不同的數集。這些符號是數學中常用的集合符號，各自的定義如下：

  1. 𝑁 - 自然數集 set of all nonnegative integers.
    定義：自然數集包含所有正整數，有時包括零（取決於定義）。，例如 𝑁 = { 0 , 1 , 2 , 3 , … } N={0,1,2,3,…} 或 𝑁 = { 1 , 2 , 3 , … } N={1,2,3,…}。用途：用於表示計數和序列的基本數字。
  2. 𝑍 - 整數集 set of all integers (both positive and negative)
    定義：整數集包含所有正整數、負整數和零，即 𝑍 = { … , − 3 , − 2 , − 1 , 0 ,1 , 2 , 3 , … } Z={…,−3,−2,−1,0,1,2,3,…}。用途：用於表示沒有小數或分數的數字，適用於計數、排序和一些代數操作。
  3. 𝑄 - 有理數集 set of all rational numbers
    定義：有理數集包含所有可以表示為兩個整數之比的數字，其中分母不為零，即 𝑄 = { p q \frac{p}{q} ∣p,q∈ Z,q ≠ 0 }。 用途：用於表示可以用分數表示的數字。
  4. 𝑅 - 實數集 set of all real numbers
    定義：實數集包含所有可以在數線上找到的數字，包括有理數和無理數，即 𝑅 。 用途：用於表示連續的數字範圍，包括所有的整數、分數、小數和無理數（如 π 和 √2 ）。
  5. 𝐶 - 複數集 set of all complex number
    定義：複數集包含所有形如 𝑎 + 𝑏 𝑖 的數字，其中 𝑎 和 𝑏 是實數， 𝑖 是虛數單位，滿足 𝑖² = − 1 ，即 𝐶 = { 𝑎 + 𝑏 𝑖 ∣ 𝑎 , 𝑏 ∈ 𝑅 } 。 用途：用於表示包含實數和虛數部分的數字，適用於複雜的數學分析和工程應用。
