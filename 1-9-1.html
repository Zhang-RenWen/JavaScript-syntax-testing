1-9-1 Formal Logic

Statement(Proposition) And Connective

在離散數學中，"Statement" 通常指的是一個可以被判定為真或假的陳述句。這些陳述句是邏輯推理和證明的基礎。以下是一些關於 statement 的基本概念：

  1. 命題(Proposition or Statement) ：這是一個可以被判定為真或假的陳述句。例如：

    • "2 是一個偶數。"（真）
    • "5 是一個偶數。"（假）

  2. 複合命題（Compound Proposition） ：由一個或多個命題組成的陳述句，並且使用邏輯連接詞（如 "和"、"或"、"非"）來組合。例如：

    • "2 是一個偶數且 3 是一個奇數。"（真）
    • "2 是一個偶數或 5 是一個偶數。"（真）

  3. 邏輯連接詞（Logical Connectives） ：用於構建複合命題的連接詞，包括：

    • AND (∧)：和
    • OR (∨)：或
    • NOT (¬)(')：非
    • IMPLIES(Implication) (→)：蘊涵
    • Equivalence (↔)：雙向蘊涵

  4. 真值表（Truth Table） ：用來顯示命題在所有可能情況下的真值。例如，對於命題 p 和 q ，可以構建如下的真值表：

    P Q P ∧ Q P ∨ Q ¬P
    True True True True False
    True False False True False
    False True False True True
    False False False False True
  • IMPLIES (P → Q) 的值為真，若 P 則 Q。
  • Equivalence (P ↔ Q) is the same as (P → Q)∧(Q → P)。

這些概念構成了離散數學中對命題邏輯和證明方法的基礎。

Negation

在邏輯學中，否定（Negation）是一個基本的邏輯運算符，通常表示為 ¬ 或 ∼。否定運算符的作用是將一個命題的真值取反，即如果原命題為真，則否定後的命題為假；如果原命題為假，則否定後的命題為真。

德摩根律 (De Morgan's Law)

德摩根律 (De Morgan's Law) 是邏輯學中的重要定理，用於描述 AND 和 OR 運算的否定形式。德摩根律有兩個主要定理：

  1. ¬ ( p ∧ q ) ≡ ( ¬ p ) ∨ ( ¬ q ) \neg (p \land q) \equiv (\neg p) \lor (\neg q)

    • 否定 AND 運算，相當於將每個運算元取否並使用 OR 運算。

  2. ¬ ( p ∨ q ) ≡ ( ¬ p ) ∧ ( ¬ q ) \neg (p \lor q) \equiv (\neg p) \land (\neg q)

    • 否定 OR 運算，相當於將每個運算元取否並使用 AND 運算。

證明 p → q p \rightarrow q 不等於 q → p q \rightarrow p

為了證明 p → q p \rightarrow q 不等於 q → p q \rightarrow p ，我們可以使用真值表來展示這兩個命題在不同情況下的真值結果不同。

首先，我們構建 p → q p \rightarrow q 和 q → p q \rightarrow p 的真值表：

p q p → q p \rightarrow q q → p q \rightarrow p
True True True True
True False False True
False True True False
False False True True

分析

從真值表中可以看出，當 p p 和 q q 的值如下時：

  1. p p 為 True， q q 為 True：

    • p → q p \rightarrow q 為 True
    • q → p q \rightarrow p 為 True

  2. p p 為 True， q q 為 False：

    • p → q p \rightarrow q 為 False
    • q → p q \rightarrow p 為 True

  3. p p 為 False， q q 為 True：

    • p → q p \rightarrow q 為 True
    • q → p q \rightarrow p 為 False

  4. p p 為 False， q q 為 False：

    • p → q p \rightarrow q 為 True
    • q → p q \rightarrow p 為 True

可以觀察到在以下兩種情況下， p → q p \rightarrow q 與 q → p q \rightarrow p 的真值結果不同：

  • 當 p p 為 True 而 q q 為 False 時， p → q p \rightarrow q 為 False，而 q → p q \rightarrow p 為 True。
  • 當 p p 為 False 而 q q 為 True 時， p → q p \rightarrow q 為 True，而 q → p q \rightarrow p 為 False。

結論

由此，我們可以得出結論： p → q p \rightarrow q 不等於 q → p q \rightarrow p ，因為在不同的情況下，它們的真值結果不同。這證明了 p → q p \rightarrow q 和 q → p q \rightarrow p 不是等價的邏輯表達式。

Negation Implication connective

As we discussed before, the compound statement of implication connective is always true when The hypothesis is false, or The result is true. Therefore, we know that:

(A→B) ↔ (A' ∨ B)

From this conclusion, we take negation on both sides, then we get:

(A→B)' ↔ (A' ∨ B)'

Now, implement De Morgan's Law, we get:

(A→B)' ↔ A ∧ B'

* This result complies with the truth table of implication connective, as we know that the only situation for implication connective statement to be false is when A is true, and B is false.

Negation of Equivalence Connective

As we know that:

(A↔B) ↔ (A→B) ∧ (B→A)

Then, we can take negation on both sides:

(A↔B)' ↔ [(A→B) ∧ (B→A)]'

Now, implement De Morgan's Law, we get:

(A↔B)' ↔ (A→B)' ∨ (B→A)'

After Negation

(A↔B)' ↔ (A∧B') ∨ (B∧A')

The conclusion is that, for A is not equivalent to B, it would be A is true and B is false or B is true and A is false. This means that, in order for A → B to be false, A and B must have different truth values.

常見重言式 Tautologies

在邏輯學中，重言式（Tautologies）是指在任何情況下都為真的邏輯命題或公式。以下是一些常見的重言式：

  1. 重言律 (Tautology Law)

    • p ∨ ¬ p p \lor \neg p
    • 不管 p p 是真還是假，這個命題總是為真。

  2. 排中律 (Law of Excluded Middle)

    • p ∨ ¬ p p \lor \neg p
    • 任意命題 p p 或它的否定總是為真。

  3. 同一律 (Law of Identity)

    • p → p p \rightarrow p
    • 任意命題 p p 總是蘊涵它自身。

  4. 雙重否定律 (Double Negation Law)

    • ¬ ( ¬ p ) ≡ p \neg (\neg p) \equiv p
    • p p 的雙重否定等於 p p 本身。

  5. 吸收律 (Absorption Law)

    • p ∨ ( p ∧ q ) ≡ p p \lor (p \land q) \equiv p
    • p ∧ ( p ∨ q ) ≡ p p \land (p \lor q) \equiv p
    • 任意命題 p p 和 q q 總是可以吸收成 p p 本身。

  6. 冪等律 (Idempotent Law)

    • p ∨ p ≡ p p \lor p \equiv p
    • p ∧ p ≡ p p \land p \equiv p
    • 任意命題 p p 和自身的或（與）運算等於 p p 本身。

  7. 重言推理律 (Tautological Implication)

    • ( p → q ) ∨ ( q → p ) (p \rightarrow q) \lor (q \rightarrow p)
    • 不管 p p 和 q q 是真還是假，這個命題總是為真。

  8. 分配律 (Distribution Law)

    • p ∨ ( q ∧ r ) ≡ ( p ∨ q ) ∧ ( p ∨ r ) p \lor (q \land r) \equiv (p \lor q) \land (p \lor r)
    • p ∧ ( q ∨ r ) ≡ ( p ∧ q ) ∨ ( p ∧ r ) p \land (q \lor r) \equiv (p \land q) \lor (p \land r)
    • 任意命題 p p 、 q q 和 r r 總是遵循分配律。

這些重言式在邏輯推理和證明中非常有用，因為它們提供了確保命題在任何情況下都為真的結構。

基本邏輯定律

以下是一些常見的邏輯定律：

  1. 身份律 (Identity Law)

    • p ∧ True ≡ p p \land \text{True} \equiv p
    • p ∨ False ≡ p p \lor \text{False} \equiv p

  2. 支配律 (Domination Law)

    • p ∨ True ≡ True p \lor \text{True} \equiv \text{True}
    • p ∧ False ≡ False p \land \text{False} \equiv \text{False}

  3. 雙重否定律 (Double Negation Law)

    • ¬ ( ¬ p ) ≡ p \neg (\neg p) \equiv p

  4. 冪等律 (Idempotent Law)

    • p ∨ p ≡ p p \lor p \equiv p
    • p ∧ p ≡ p p \land p \equiv p

  5. 交換律 (Commutative Law)

    • p ∨ q ≡ q ∨ p p \lor q \equiv q \lor p
    • p ∧ q ≡ q ∧ p p \land q \equiv q \land p

  6. 結合律 (Associative Law)

    • ( p ∨ q ) ∨ r ≡ p ∨ ( q ∨ r ) (p \lor q) \lor r \equiv p \lor (q \lor r)
    • ( p ∧ q ) ∧ r ≡ p ∧ ( q ∧ r ) (p \land q) \land r \equiv p \land (q \land r)

  7. 分配律 (Distributive Law)

    • p ∧ ( q ∨ r ) ≡ ( p ∧ q ) ∨ ( p ∧ r ) p \land (q \lor r) \equiv (p \land q) \lor (p \land r)
    • p ∨ ( q ∧ r ) ≡ ( p ∨ q ) ∧ ( p ∨ r ) p \lor (q \land r) \equiv (p \lor q) \land (p \lor r)

  8. 吸收律 (Absorption Law)

    • p ∨ ( p ∧ q ) ≡ p p \lor (p \land q) \equiv p
    • p ∧ ( p ∨ q ) ≡ p p \land (p \lor q) \equiv p

補充名詞解釋

  1. Well-Formed Formula（WFF） - 在邏輯學和數理邏輯中，良構公式（Well-Formed Formula, WFF）是根據特定語法規則構建的語句或表達式。這些公式被認為是語法上正確的，可以用來進行邏輯推理和證明。良構公式的定義依賴於所使用的邏輯系統，比如命題邏輯或一階邏輯。
  2. 有效論證（valid argument） -在邏輯學中，有效論證（valid argument）是一種論證形式，其中，如果所有前提都為真，則結論必然為真。有效論證確保了從前提到結論的邏輯推導過程是正確的，即不存在使前提為真而結論為假的可能情況。一個論證由一組前提和一個結論組成。如果根據前提，能夠邏輯必然地導出結論，則該論證是有效的。形式化地說，如果論證的前提集合為 { 𝑃 1 , 𝑃 2 , … , 𝑃 𝑛 } ，結論為 𝐶

deduction 演繹法

演繹法（Deduction）是從一般原理出發，通過邏輯推理得出特定結論的一種方法。在形式邏輯中，自然演繹法是一種常用的證明技術。以下是如何使用自然演繹法來證明一個論證有效的步驟和具體例子。

自然演繹法的步驟

  • 列出前提：寫下所有的前提。
  • 應用推理規則：從前提出發，使用邏輯推理規則逐步推導。
  • 得出結論：通過一系列推理步驟，最終得出結論。

常見的推理規則包括：

  • Modus Ponens (MP): 𝑃 → 𝑄 , 𝑃 ⊨ 𝑄
  • Modus Tollens (MT): 𝑃 → 𝑄 , ¬ 𝑄 ⊨ ¬ 𝑃
  • P→Q,¬Q⊨¬P 合取導出 (Conjunction Introduction): 𝑃 , 𝑄 ⊨ 𝑃 ∧ 𝑄
  • 合取消去 (Conjunction Elimination): 𝑃 ∧ 𝑄 ⊨ 𝑃 P∧Q⊨P, 𝑃 ∧ 𝑄 ⊨ 𝑄
  • 析取導出 (Disjunction Introduction): 𝑃 ⊨ 𝑃 ∨ 𝑄 P⊨P∨Q
  • 假言三段論 (Hypothetical Syllogism): 𝑃 → 𝑄 , 𝑄 → 𝑅 ⊨ 𝑃 → 𝑅

自然演繹法的例子

讓我們使用自然演繹法來證明以下論證：

命題： ( 𝑃 → 𝑄 ) , ( 𝑄 → 𝑅 ) ⊨ ( 𝑃 → 𝑅 )

證明：

  1. P→Q （前提）
  2. Q→R （前提）
  3. 假設 𝑃 （假設，用於條件導出）
  4. 根據 (1) 和 (3)， 𝑄 （使用 Modus Ponens，1, 3）
  5. 根據 (2) 和 (4)， 𝑅 （使用 Modus Ponens，2, 4）
  6. 因此， 𝑃 → 𝑅 （條件導出，3-5）

這證明了 ( 𝑃 → 𝑄 ) , ( 𝑄 → 𝑅 ) ⊨ ( 𝑃 → 𝑅 ) 。
