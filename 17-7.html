17-7 Official shebang support

shebang 是老式的 Unix 對話，表示主題標籤，後跟感嘆號： #! （其中“bang”是“！”的俚語）。自古以來，文件開頭的註釋都是以 #! 告訴 shell 你這裡有一個可執行腳本，以及使用什麼引擎來運行它。例如，清單下列是一個典型的 bash 腳本。 


#!/bin/bash

echo "Hello, world!"

您可以使用 ./hello.sh 直接運行類似於清單上面範例中的文件。 您可以使用 JavaScript 執行類似的操作，如下範例所示。 


#!/usr/bin/env node

console.log("Hello, world!");

上面範例告訴操作系統使用節點程序來運行該腳本。現在，您只需輸入 ./hello.js 即可運行它。沒有 #! ，這是行不通的。

Shebang support 是規範中的功能更新之一，已在多個環境中實現並非正式採用。
