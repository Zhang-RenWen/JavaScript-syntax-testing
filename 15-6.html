15-6 Logical OR assignment(||=)

Logical OR assignment ||= 左側的值若為 Falsy ,就會將 ||= 右側的值賦予 ||= 左側，並回傳賦值運算後的結果。 


const a = { duration: 50, title: '' };

a.duration ||= 10;
console.log(a.duration);
// Expected output: 50

a.title ||= 'title is empty.';
console.log(a.title);
// Expected output: "title is empty"

語法


x ||= y

下列舉另外一種寫法表示，方便理解本章運算子的義涵。


x || (x = y);

x 為 Truthy 的話， ||= 左側的部份不會被賦值。 


const x = 1;
x ||= 2;
// Expected output: 1

配合物件修飾符 get/set 的範例


const x = {
  get value() {
    return 1;
  },
  set value(v) {
    console.log("Setter called");
  },
};

x.value ||= 2; // Expected output: 1

若 ||= 左側為 Truthy ，除了不會賦值外，事實上 ||= 右側的程式連執行都不會發生。 


const x = 1;
x ||= console.log("y evaluated");
// Logs nothing

更多示範案例加深理解：


document.getElementById("lyrics").textContent ||= "No lyrics.";
