1-1-3 實現置中

首先看以下範例


<div>
  <div class="wp">
    <div class="box fixed-size">text</div>
  </div>
</div>
        


.wp {
  border: 1px solid red;
  width: 300px;
  height: 300px;
}
.box {
  background: green;
}
.box.fixed-size {
  width: 100px;
  height: 100px;
}
text

下面列舉3種方式將以上範例置中

置中元素定長寬

在置中元素定長寬的情況下，有以下3種實現置中顯示的方法。

1. absolute + 負 margin


<div>
  <div class="wp wp-type1">
    <div class="box fixed-size box-type1">text</div>
  </div>
</div>
        


.wp-type1 {
  position: relative;
}
.box-type1 {
  position: absolute;
  top: 50%;
  left: 50%;
  margin-left: -50px;
  margin-top: -50px;
}
text

2. absolute + margin auto


<div>
  <div class="wp wp-type2">
    <div class="box fixed-size box-type2">text</div>
  </div>
</div>
        


.wp-type2 {
  position: relative;
}
.box-type2 {
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
}
text

3. absolute + calc


<div>
  <div class="wp wp-type3">
    <div class="box fixed-size box-type3">text</div>
  </div>
</div>
        


.wp-type3 {
  position: relative;
}
.box-type3 {
  position: absolute;
  top: calc(50% - 50px);
  left: calc(50% - 50px);
}
text
