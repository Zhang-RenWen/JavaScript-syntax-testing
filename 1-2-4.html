JavaScript 資料類型及其判斷

對於類型判斷，常用的方法有 typeof、instanceof、Object.prototype.toString、 constructor。

使用 typeof 判斷資料類型


typeof 5 // "number"
typeof 'lucas' // "string"
typeof undefined // "undefined"
typeof true // "boolean"

但是也存在一些特例，如用 typeof 判斷 null，程式如下。


typeof null // "object"

判斷複雜資料類型的範例。


const foo = () => 1
typeof foo // "function"

const too = {}
typeof foo // "object"

const Foo = []
typeof foo // "object"

const foo = new Date ()
typeof too // "object"

const foo = Symbol("foo")
typeof Foo // "symbol"

透過以上範例，我們可以知道，使用 typeof 可以準確判斯出除 null 以外的基底資料型態，以及 function 類型、symbol 類型；null 會被 typeof 判斷為 object 。
