2-2 箭頭函式

2-2-1 語彙範圍

  • 無法被給定名稱只能指派給變數(隱性命名)
  • 沒有自己的 語彙範圍
  • 沒有自己的 this
  • 不會建立閉包
  • 箭頭函式沒有建構函式

function puzzle(){
    return function(){
      console.log(arguments)
    }
}
puzzle('a','b','c')(1,2,3) // [1,2,3]


function puzzle(){
  return ()=>{ console.log(arguments)}
}
puzzle('a','b','c')(1,2,3) // ['a','b','c']


        

let obj = {
  normal:function(){
    console.log(this)
  },
arrow:()=>{
    console.log(this)
  }
}
obj.normal() // obj 指向調用函式自身的物件
obj.arrow() // window  由於箭頭函式沒有自己的 this 箭頭函式會向外找尋外層作用域的 this 為何

2-2-2 縮寫

箭頭後面去除 {} 可以直接接上返回值,省略 return。注意需要寫在箭頭同一行。 

// before ES6
function(a,b){
  return a + b
}

// after ES6
(a,b)=> a + b

// 單個引數，小括號也可省略
a=>a*a

其餘參數


function fn() {
  console.log(arguments)
}
fn(1, 2, 3, 4) //Arguments(4) [1, 2, 3, 4,]


const fn = () => {
  console.log(arguments)
}
fn(1, 2, 3, 4) // VM1730:2 Uncaught ReferenceError: arguments is not defined
