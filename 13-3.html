13-3 String.prototype.trimStart()

trimStart() 方法會刪除字串開頭的空白字符。trimLeft() 是此方法的別名。


const greeting = '   Hello world!   ';

console.log(greeting);
// Expected output: "   Hello world!   ";

console.log(greeting.trimStart());
// Expected output: "Hello world!   ";

別名


trimStart()

trimLeft()

為了與 String.prototype.padStart 等函數保持一致，標準方法名稱為 trimStart。但是，出於 Web 兼容性原因，trimLeft 仍然是 trimStart 的別名。在某些引擎中，這意味著： 


String.prototype.trimLeft.name === "trimStart";

範例


let str = "   foo  ";

console.log(str.length); // 8

str = str.trimStart();
console.log(str.length); // 5
console.log(str); // 'foo  '
